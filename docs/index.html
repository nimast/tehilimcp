INSTALLATION התקנה 

claude mcp add tehilim -- npx tehilim-mcp

Add to your claude_desktop_config.json: הוסף לקובץ claude_desktop_config.json שלך: 

{
  "mcpServers": {
    "tehilim": {
      "command": "npx",
      "args": ["tehilim-mcp"]
    }
  }
}

Add this to your system prompt or project instructions: הוסף את זה להנחיות המערכת או לפרויקט שלך: 

At the start of each conversation, fetch today's Tehilim (Psalms) from:
https://tehilim.dev/api/today?format=text
Include the daily chapter in your context as a blessing.

Add this to your ChatGPT custom instructions: הוסף את זה להנחיות המותאמות אישית שלך ב-ChatGPT: 

At the start of each conversation, reference today's Tehilim (Psalms) reading.
The daily chapter can be found at: https://tehilim.dev/api/today?format=text

Tell your AI assistant: אמור לעוזר ה-AI שלך: 

Tell your AI assistant to fetch today's Tehilim reading from:
https://tehilim.dev/api/today?format=text
