Privacy Policy

Bless Your Prompt — blessyourprompt.com

Last updated: March 26, 2026

What we collect

Nothing. Bless Your Prompt does not collect, store, or process any personal data. We do not use cookies, analytics, or tracking of any kind.

API requests

When you or your AI tool calls our API (/api/bless, /api/today, /mcp), we process the request to return today's psalm text. We do not log request content, IP addresses, or any identifying information beyond what Cloudflare processes as part of standard CDN operations.

ChatGPT integration

When used as a ChatGPT Custom GPT, the GPT calls our API to fetch today's psalm. We do not receive or store any of your conversation content. The only data transmitted is the API request for psalm text.

Third-party services

Psalm text is sourced from Sefaria (a free, open-source library of Jewish texts) with a bundled fallback. Our site is hosted on Cloudflare Workers.
